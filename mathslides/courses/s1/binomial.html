S1 Binomial Distribution

Objectives:

  • Understand the notation $X \sim B(n,p)$
  • Use the formula to work out probabilities
  • Use the tables to work out cumulative probabilities
  • Deal with inequality signs in cumulative probabilities
  • Deal with $p>0.5$ by considering $q=1-p$
  • Learn the formulas for the mean and standard deviation of a binomial distribution

Inverse Events

You may have noticed that your tables only go as far as $p=0.5$. What happens if $p>1.5$?

In order to deal with this, we must instead consider the inverse event with probability $q = 1-p$. Fortunatley, if $p>0.5$ then $q < 0.5$ and we can use our tables again!

However, we must be careful! If in $10$ trials we have $7$ successes, then there must be $3$ failures.

Translating to failure

If $X$ is the number of successful trials and $$ X \sim B(n,p) $$ then we can let $Y$ be the number of failures, so that $$ Y \sim B(n,1-p) $$ The one thing to be careful about it changing the information in the question so that it refers to failures instead.

The best method

In general, if there are $k$ successes then there must be $n-k$ failures. We can use this to translate a statement about successes into a statement about failures. If there are $20$ trials and we are interested in $X \leq 12$ successes, then listing the possible outcomes gives: $$ X = 0, 1, 2, \ldots, 11, 12 $$ $$ Y = 20, 19, 18, \ldots, 9, 8 $$ and we have $ Y \geq 8 $ instead. If $p>0,5$ you should always list the outcomes in this way.
